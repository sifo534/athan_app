# Audio Assets

Ce dossier contient les fichiers audio pour l'application Athan Muslim.

## Structure des fichiers

```
audio/
├── athan/
│   ├── athan_abdul_basit.mp3
│   ├── athan_mishary_rashid.mp3
│   └── athan_maher_al_mueaqly.mp3
├── quran/
│   ├── al_fatihah_abdul_basit.mp3
│   ├── al_baqarah_abdul_basit.mp3
│   └── ...
├── dhikr/
│   ├── subhanallah.mp3
│   ├── alhamdulillah.mp3
│   └── allahu_akbar.mp3
└── notification.mp3
```

## Sources recommandées

Pour obtenir des fichiers audio authentiques :

1. **Athan** : Enregistrements officiels des mosquées
2. **Coran** : Récitations de Haramain, Quran.com
3. **Dhikr** : Enregistrements de qualité islamique

## Format recommandé

- **Format** : MP3 ou AAC
- **Qualité** : 128 kbps minimum
- **Durée** : Athan (3-5 min), Dhikr (10-30 sec)

## Note importante

Assurez-vous que tous les fichiers audio respectent les droits d'auteur et sont utilisés avec permission appropriée.
